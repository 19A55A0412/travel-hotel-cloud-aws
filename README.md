# Cloud-Based Travel & Hotel Management System

## MSc Cloud Computing Project â€“ University of East London

This project presents a cloud-based, AI-powered Travel and Hotel Management System designed to improve booking efficiency, enhance customer experience, and provide scalable and secure operations using AWS cloud services. The system leverages serverless architecture and intelligent automation to deliver real-time booking, predictive analytics, and personalized services.

---

## Project Overview
The travel and hospitality industry faces challenges such as manual booking, inefficient operations, and lack of personalization. This solution addresses these problems by implementing a modern cloud architecture using AWS to ensure high availability, security, and scalability.

This system automates booking, manages hotel operations, and provides personalized recommendations using AI and predictive analytics.

---

## Cloud Architecture
The system uses a scalable and secure AWS architecture.

<img width="1051" height="663" alt="image" src="https://github.com/user-attachments/assets/655a0757-ff25-4aab-836c-df07a8cab5d5" />



Key components include:
- Route 53 for DNS and traffic routing
- CloudFront for global content delivery
- Elastic Load Balancer for distributing traffic
- EC2 for application hosting
- S3 for static content and storage
- DynamoDB for fast NoSQL data storage
- Lambda for serverless processing
- CloudWatch for monitoring and logging
- IAM for security and access control
